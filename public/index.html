Relink Folder Import

Wskaż folder z muzyką. Nic nie wysyłamy z dysku — do backendu leci tylko lista nazw plików. Backend wyszukuje utwory na Spotify, deduplikuje i tworzy playlistę.

1) Wybierz folder z audio

Wspierane przeglądarki: Chrome/Edge/Brave/Opera (API: showDirectoryPicker). Skanowanie jest rekurencyjne. Rozszerzenia: mp3, m4a, aac, flac, wav, ogg.

#Relatywna ścieżka

2) Tworzenie playlisty

Pod maską: backend czyści tytuły („official video”, „out now”, itp.), zgaduje artist/title z nazwy pliku, szuka na Spotify, liczy podobieństwo i deduplikuje po track_id.

Backend: http://localhost:5174
